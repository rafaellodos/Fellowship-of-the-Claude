## WHAT THIS IS

Aragorn holds active project state — things currently in motion with open decisions. This is not a log of everything; it is the minimum necessary to pick up where you left off.

Write to Aragorn at session close. Clear entries when projects close. Archive to `memory/Aragorn/archive-[date].md` when this file grows beyond 200 lines.

---

## HOW TO WRITE TO ARAGORN

Append a session block at the bottom of this file:

```
## SESSION [YYYY-MM-DD]
Active workspace: [which workspace]
What happened: [1-2 sentences — what was worked on]
Decisions: [notable decisions made]
Open threads: [unresolved items to pick up next session]
```
